# Tools & Resources

This page lists the primary tools used by Product 
Associate Interns at HotWax Commerce.

## Figma
**Purpose:** UI/UX Design  
**PA Use Case:** Reviewing wireframes and annotating 
design specs for new features.

**Access:** Ask your manager for the HotWax Figma workspace link.

## GitHub
**Purpose:** Repository Management  
**PA Use Case:** Tracking issues, pull requests, and 
syncing documentation across teams.

**Repository:** https://github.com/bhardwajdiv/hotwax-docs

## GitBook
**Purpose:** Documentation Platform  
**PA Use Case:** Publishing and maintaining SOPs, 
user guides, and internal knowledge bases.

**Space:** This documentation you are reading right now!

## HotWax OMS Docs
**Official Documentation:** https://docs.hotwax.co
