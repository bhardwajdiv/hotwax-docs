# Getting Started

Welcome to HotWax Commerce! This guide will help you 
get set up as a Product Associate Intern.

## Prerequisites
- GitHub account
- GitBook access
- Figma account

## Setup Steps

### 1. Access HotWax OMS
Log in to the HotWax OMS dashboard using credentials 
provided by your manager.

### 2. Explore the Documentation
Visit the HotWax docs at:
https://docs.hotwax.co

### 3. Set Up Your Tools
- **Figma** — For reviewing UI/UX designs
- **GitHub** — For managing repositories and issues
- **GitBook** — For reading and writing documentation

- [ ] Complete onboarding session
- [ ] Set up local development environment
- [ ] Read Transfer Orders documentation
